# Backend Developer Assignment

## How to run

1. Install packages

`$ npm install`

1. Build project

`$ npm run build`

1. Initialize database

`$ npm run db:init`

1. Start the server

`$ npm run start`

The server will be running on port 3000

## Database

* Create tables

`$ npm run db:init`

* Populate database with task example data

`$ npm run db:populate`

* Drop tables and create again

`$ npm run db:reset`

* Delete tables

`$ npm run db:delete`
